# FakeFriends

A ruby gem for generating consistent fake user data (for demoing social networking apps.)

## Installation

Add this line to your application's Gemfile:

    gem 'fake_friends'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install fake_friends
